STEP 1: Copy Embed

Go to the Youtube video you wish to embed, and click share. Select Embed, and copy the text shown on screen.

STEP 2: Paste Embed

Open your .html file, and paste the embed code in the \ tag.

STEP 3: Create a Div

Create a div with a custom class, containing the embed code you just pasted.

STEP 4: Create a Stylesheet to format your video

Make a new .css stylesheet, and put all formatting for the video in the class you created for the div.
